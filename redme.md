# SmartERP - Billing, Inventory & Accounting System

## 📌 Project Description
SmartERP is a web-based ERP system built using Flask.  
It helps manage company operations like sales, purchase, inventory, customers, suppliers, and financial year tracking.

## 🚀 Features
- Company Management
- Customer & Supplier Management
- Product Inventory System
- Sales & Purchase Modules
- Invoice Generation
- Financial Year Switching
- Dashboard Overview
- Reports Section

## 🛠️ Tech Stack
- Python (Flask)
- SQLite / SQLAlchemy
- HTML, CSS, Bootstrap
- Jinja2 Templates
